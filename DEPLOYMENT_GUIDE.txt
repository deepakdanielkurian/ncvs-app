  NCVS DONATION TRACKER — COMPLETE DEPLOYMENT GUIDE

This is a React + Firebase app. Follow these steps EXACTLY.
Total time: about 30–45 minutes.

───────────────────────────────────────────────────────────
STEP 1 — Install Node.js on your computer
───────────────────────────────────────────────────────────
1. Go to: https://nodejs.org
2. Download the LTS version (the green button)
3. Install it (click Next → Next → Finish)
4. Open Terminal (Mac) or Command Prompt (Windows)
5. Type: node --version
   You should see something like: v20.0.0
   If yes, Node.js is installed correctly.

───────────────────────────────────────────────────────────
STEP 2 — Create Firebase project
───────────────────────────────────────────────────────────
1. Go to: https://console.firebase.google.com
2. Sign in with your Google account
3. Click "Add project"
4. Name it: ncvs-donation-tracker
5. Disable Google Analytics (not needed) → Create project
6. Wait for it to create (about 30 seconds)

── Enable Authentication ──
1. In your project, click "Authentication" in left menu
2. Click "Get started"
3. Click "Email/Password"
4. Toggle "Enable" to ON
5. Click Save

── Create Firestore Database ──
1. Click "Firestore Database" in left menu
2. Click "Create database"
3. Select "Start in production mode"
4. Choose location: asia-south1 (Mumbai, closest to Kerala)
5. Click Enable

── Get your Firebase config ──
1. Click the gear icon ⚙️ next to "Project Overview"
2. Click "Project settings"
3. Scroll down to "Your apps" section
4. Click the </> Web icon to add a web app
5. Name it: ncvs-app → click Register app
6. You will see a code block like this:
   const firebaseConfig = {
     apiKey: "AIzaSy...",
     authDomain: "ncvs-....firebaseapp.com",
     projectId: "ncvs-...",
     ...
   }
7. COPY this entire config block — you need it in Step 3

───────────────────────────────────────────────────────────
STEP 3 — Set up the app code
───────────────────────────────────────────────────────────
1. Extract this ZIP file on your Desktop
2. Open the ncvs-firebase folder
3. Open the file: src/firebase.js
4. Replace the placeholder values with YOUR config from Step 2:

   CHANGE THIS:
   apiKey: "YOUR_API_KEY",
   authDomain: "YOUR_PROJECT.firebaseapp.com",
   projectId: "YOUR_PROJECT_ID",
   ...

   TO YOUR ACTUAL VALUES:
   apiKey: "AIzaSyXXXXXXXXXXX",
   authDomain: "ncvs-donation-tracker.firebaseapp.com",
   projectId: "ncvs-donation-tracker",
   ...

5. Save the file

───────────────────────────────────────────────────────────
STEP 4 — Install and build the app
───────────────────────────────────────────────────────────
1. Open Terminal / Command Prompt
2. Navigate to the app folder:
   cd Desktop/ncvs-firebase
3. Install packages (downloads React, Firebase etc.):
   npm install
   (Wait 1–2 minutes, you'll see lots of text)
4. Build the app:
   npm run build
   (Wait 30 seconds, creates a "dist" folder)

───────────────────────────────────────────────────────────
STEP 5 — Deploy to Firebase Hosting (FREE)
───────────────────────────────────────────────────────────
1. Install Firebase tools:
   npm install -g firebase-tools
2. Login to Firebase:
   firebase login
   (Opens browser — sign in with Google)
3. Initialize hosting:
   firebase init hosting
   Answer the questions:
   → "Which Firebase project?" → select ncvs-donation-tracker
   → "Public directory?" → type: dist
   → "Single-page app?" → type: y
   → "Overwrite dist/index.html?" → type: N
4. Deploy:
   firebase deploy --only hosting
5. You get a URL like:
   https://ncvs-donation-tracker.web.app
   ✅ DONE! Your app is live.

───────────────────────────────────────────────────────────
STEP 6 — Create your admin account
───────────────────────────────────────────────────────────
1. Open your app URL in browser
2. Go to Firebase console → Authentication → Users
3. Click "Add user"
4. Enter your email and a strong password
5. Click Add user
6. Now go to Firestore Database → Start collection
7. Collection ID: users
8. Document ID: (paste the UID shown next to your email in Authentication)
9. Add these fields:
   email   → string → your@email.com
   role    → string → admin
   name    → string → Your Name
10. Click Save
11. Log in to the app with your email and password
    You will have full admin access!

── Create treasurer account ──
1. In the app, go to Admin → Users
2. Fill in the treasurer's email, password, name
3. Select role: treasurer
4. Click Create user
5. Share the email and password with your treasurer

───────────────────────────────────────────────────────────
STEP 7 — Install app on phone
───────────────────────────────────────────────────────────
iPhone (Safari only):
1. Open your app URL in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Name: NCVS Tracker → Tap Add
✅ App icon appears — opens fullscreen!

Android (Chrome):
1. Open URL in Chrome
2. Tap 3-dot menu → "Add to Home screen" or "Install app"
3. Tap Add
✅ App icon appears on home screen!

───────────────────────────────────────────────────────────
UPDATING THE APP LATER
───────────────────────────────────────────────────────────
When you receive an updated app from Claude:
1. Replace files in your ncvs-firebase folder
2. Open terminal in that folder
3. Run: npm run build
4. Run: firebase deploy --only hosting
5. App updates in about 30 seconds — same URL

───────────────────────────────────────────────────────────
ALTERNATIVELY — DEPLOY TO VERCEL (also free)
───────────────────────────────────────────────────────────
1. Go to: https://vercel.com → Sign up with GitHub
2. Install Vercel CLI: npm install -g vercel
3. In terminal, inside ncvs-firebase folder:
   vercel
4. Follow the prompts
5. You get URL like: ncvs-firebase.vercel.app

For auto-deploy on every update:
1. Push code to GitHub
2. Connect GitHub repo to Vercel
3. Every push → auto-deploys!

───────────────────────────────────────────────────────────
DATA STORAGE
───────────────────────────────────────────────────────────
All data is stored in Firebase Firestore (Google's cloud).
- Syncs instantly between your phone and treasurer's phone
- Works offline — data syncs when back online
- Data is NOT stored on any device — it's in the cloud
- Never lost even if you change phone

Collections in Firestore:
  entries    — All donations and subscriptions
  members    — Member contact list with phone/WhatsApp
  attendance — Session attendance records
  users      — User roles (admin/treasurer)
